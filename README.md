# STM32 Bare-Metal Driver Development & Custom PCB Design

## 📌 Overview

This project demonstrates low-level embedded systems development using STM32 microcontrollers.

## 🚀 Features

- UART Driver (Polling, Interrupt, DMA)
- ADC Driver (Polling + DMA)
- Timer-triggered ADC sampling
- Custom STM32 PCB Design

## 🧠 Key Concepts

- Register-level programming
- RCC configuration
- Interrupt handling (NVIC)
- DMA optimization
- Power design and decoupling

## 🛠 Tools Used

- STM32CubeIDE
- KiCad
- Git & GitHub
